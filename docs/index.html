How it works
Four steps. Zero babysitting.
Submit a task. The Overseer decomposes it, routes to the right agents, and verifies the output before reporting back.
1

Submit

Send a task via API, GitHub issue, or Telegram. Plain English.

2

Decompose

Overseer breaks the task into subtasks and picks the cheapest capable agent for each.

3

Execute

Agents work in parallel with 75+ tools. Reflexion gates catch errors before they ship.

4

Deliver

Results verified, cost tracked, code committed. You get a report with what changed and why.

Reliability
4-Tier LLM Fallback

If one model fails, the next one picks up. No single point of failure.
