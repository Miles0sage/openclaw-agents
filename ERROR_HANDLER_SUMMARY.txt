================================================================================
ERROR TYPES CLASSIFICATION
================================================================================

ErrorType.TIMEOUT
  → Request exceeded time limit
  → Action: Retry with longer timeout or fallback

ErrorType.RATE_LIMIT (429)
  → Too many requests
  → Action: Exponential backoff, notify admin

ErrorType.NETWORK
  → Connection refused, reset, no route
  → Action: Retry (usually temporary)

ErrorType.AUTHENTICATION (401/403)
  → Unauthorized or forbidden
  → Action: Check credentials, don't retry

ErrorType.MODEL_ERROR
  → Model not found or invalid
  → Action: Try different model, don't retry

ErrorType.INTERNAL (500)
  → Server error
  → Action: Retry or fallback

ErrorType.UNKNOWN
  → Unclassified error
  → Action: Log and escalate
