// Detections

What we catch.

CRITICAL

SQL Injection

Database queries accepting unsanitized input — full data breach risk.

CRITICAL

Remote Code Execution

Server-side flaws allowing attackers to run arbitrary commands.

HIGH

Cross-Site Scripting (XSS)

Injected scripts stealing sessions, credentials, and user data.

HIGH

Broken Authentication

Weak login flows, session hijacking, missing rate limits.

MEDIUM

Security Misconfigurations

Exposed admin panels, default credentials, verbose error pages.

MEDIUM

Missing Security Headers

No CSP, HSTS, X-Frame-Options — easy wins most sites miss.

INFO

SSL/TLS Issues

Expired certs, weak ciphers, protocol downgrade vulnerabilities.

INFO

Open Ports & Services

Unnecessary services exposed to the internet, increasing attack surface.
